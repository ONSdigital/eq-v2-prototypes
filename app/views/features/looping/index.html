Looping features for user research

Business survey question examples

  • Collecting postcodes - business R&D - percentages at each postcode

    This design is for collecting multiple postcodes (or text input) in a loop and then collecting a percentage against each postcode.

  • Collecting text input - name of UK branch - and additional radio question

    This design is for collecting multiple text inputs in a loop and then answering an additional question against each text input.

  • Collecting a description of item and currency values - disposal and acquisition - with calculated summary

    This design is for collecting descriptions of items and currency values against them, which is then shown in calculated summaries.

  • Collecting another type of asset and a value - with calculated summary

    This design is for capturing values for defined questions, and then allowing the user to add another type of asset and collecting values for them, which is then shown in a calculated summary.

  • Long list of services - added to a calculated summary

    This design is for capturing values for defined questions, and then allowing the user to add from a long list of services and collecting values for them, which is then shown in a calculated summary.

Social survey question examples

  • Supermarket shopped at - percentage at each

    This design collects free text name for the supermarkets shopped at, and what percentage of their shopping is at each supermarket.

  • Countries travelled to - simple loop

    This design collects country name by autosuggest.


Initial looping scenarios

  • Collecting free text and yes / no

    This design collects free text information and an answer to a yes / no question.

  • Collecting postcodes and percentages

    This design allows for looping through collecting postcodes. It also then collects percentages against each postcode collected which must add up to 100%.

  • Collecting currency values and free text

    This design allows for looping through collecting free text description and currency values. A calculated summary can be then added to this design.
